# 🌸 Memory Garden Game (PWA)

An installable **Progressive Web App (PWA)** that lets users plant, explore, and personalize a mystical virtual garden. Built with AI Studio and enhanced into a fully interactive experience.

---

## ✨ Features
- 🌱 **Installable App**: Works like a native app with its own icon.
- 🌌 **Sky Navigation**: Interactive mini‑map with zoom and directional controls.
- 🔍 **Search Bar**: Quickly find flowers by name with auto‑scroll + highlight.
- 🎵 **Music Playback**: Built‑in calming tracks with play/pause toggle.
- 🛒 **Gardener’s Shops**: Buy seeds, decorations, and interactive items.
- 🌟 **Offline Support**: Service Worker caching ensures the garden loads without internet.
- 📔 **Garden Journal**: Save personal memories linked to flowers.

---

## 🚀 Run Locally

**Prerequisites:**  
- Node.js (v18+ recommended)  
- Gemini API key  

1. Install dependencies:  
   ```bash
   npm install
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
    npm run dev
4. Run the app:
   `npm run dev`
